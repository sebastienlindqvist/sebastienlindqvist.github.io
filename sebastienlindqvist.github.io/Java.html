Modded Minecraft

This is still a work in progress however, I am learning a lot in the process and I might move to C and C++ for the Windows version of Minecraft

Minecraft is a famous sandbox game that came out in 2011 and has been something I enjoyed for quite a while. Minecraft has a large community that create mods for the game either adding to or altering the game to make it more enjoyable for themselves and people who download their mods. Recently I have tried making my own mods for the game due to many of my favourite are no longer supported and can no longer be played with recent versions of the game. Readind documentations and watching tuorials have allowed me to add additional blocks within the game however, I have so much more to learn.
